Проект написан на Python 3.8.10, с использованием фраймворка Django 3.0.5.

БД - использовалась SQLite3 при разработке и PostgreSQL на боевом сервере.

Используется Docker - проект запущен в 3х контейнерах: БД, статика (Nginx) и само приложение.

Подключен GitHub Actions с тестированием на соответствие flak8, автопуш на DockerHub, автодеплой на боевой сервер и оповещение в телеграмм-бот.
